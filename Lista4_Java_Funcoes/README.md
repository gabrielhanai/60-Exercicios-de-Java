# Funções e Modularização

Lista de exercícios para praticar funções, modularização e estruturas de controle em Java.

---

## Fácil

### 1. Dobro do Valor

Crie uma função que receba um número inteiro como parâmetro e retorne o seu dobro.

### 2. Conversor de Temperatura

Escreva uma função que receba uma temperatura em Celsius e retorne o valor convertido para Fahrenheit.

### 3. Verificador de Paridade

Crie uma função que receba um número inteiro e retorne `true` se for par e `false` se for ímpar.

### 4. Média Simples

Desenvolva uma função que receba três notas (floats) e retorne a média aritmética simples.

### 5. Saudação Personalizada

Faça uma função que receba o nome de uma pessoa (string) e imprima a mensagem:

```
Olá, [nome], seja bem-vindo à aula de programação!
```

### 6. Cálculo de Área

Crie uma função que calcule e retorne a área de um retângulo, recebendo a base e a altura como parâmetros.

### 7. Maior de Dois

Escreva uma função que receba dois números e retorne o maior deles.

---

## Médio (Estruturas de Controle e Laços)

### 8. Tabuada Dinâmica

Crie uma função que receba um número e imprima a sua tabuada do 1 ao 10 utilizando um laço `for`.

### 9. Soma de Intervalo

Desenvolva uma função que receba dois números inteiros (início e fim) e retorne a soma de todos os números entre eles.

### 10. Contador de Vogais

Escreva uma função que receba uma string e, utilizando um laço de repetição, conte quantas vogais existem nela.

### 11. Calculadora Básica

Crie uma função que receba dois números e um caractere representando a operação (+, -, *, /). Utilize `switch case` para realizar o cálculo e retornar o resultado.

### 12. Validação de Senha

Faça uma função que peça ao usuário para digitar uma senha e só retorne `true` quando a senha for "1234". Use `do while`.

### 13. Fatorial

Crie uma função que calcule o fatorial de um número inteiro passado por parâmetro usando um laço `while`.

### 14. Verificador de Primos

Escreva uma função que receba um número e retorne se ele é primo ou não (utilize a lógica de divisores).

---

## Difícil (Vetores e Lógica Avançada)

### 15. Soma de Vetor

Crie uma função que receba um vetor de inteiros e seu tamanho, e retorne a soma de todos os elementos.

### 16. Busca em Vetor

Desenvolva uma função que receba um vetor de 10 posições e um valor X. A função deve retornar a posição (índice) onde X se encontra ou -1 caso não exista.

### 17. Média de Vetor com Filtro

Escreva uma função que receba um vetor de floats, calcule a média e imprima apenas os valores que estão acima dessa média.

### 18. Inversão de String

Crie uma função que receba uma string e a exiba de trás para frente.

### 19. Maior e Menor no Vetor

Faça uma função que receba um vetor de inteiros e, através de passagem de parâmetros ou exibição direta, mostre o maior e o menor valor presente.

### 20. Ordenação Simples

Desenvolva uma função que receba um vetor e o ordene em ordem crescente (pode usar o algoritmo Bubble Sort que vimos brevemente).

---

## Desafios

### 1. O Gerenciador de Notas

Crie um programa que use:

- Uma função para preencher um vetor com as notas de 5 alunos
- Outra função para calcular a média da turma
- Uma terceira para contar quantos alunos ficaram acima da média

### 2. Sistema de Login

Desenvolva uma função que valide um nome de usuário e uma senha. O usuário tem apenas 3 tentativas para acertar (use `while` ou `for`). Se falhar, o programa deve informar que a conta foi bloqueada.

### 3. Analisador de Texto

Crie uma função que receba uma frase e retorne quantas palavras ela possui (dica: conte os espaços em branco entre as palavras).

---

## Objetivos de Aprendizado

✓ Entender o conceito de funções e reutilização de código
✓ Dominar passagem de parâmetros
✓ Trabalhar com diferentes tipos de retorno
✓ Utilizar estruturas de controle (if/else, switch case)
✓ Praticar laços de repetição (for, while, do-while)
✓ Manipular vetores e strings
✓ Implementar lógica avançada com algoritmos simples
✓ Criar programas modulares e bem estruturados
